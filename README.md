# Learn German

**An interactive web app for learning the German language.**

Practice vocabulary, grammar, and everyday phrases through quizzes, flashcards, and interactive exercises — all in the browser with no sign-up required.

## Features

- Vocabulary flashcards with spaced repetition
- Grammar exercises and conjugation practice
- Common phrases and everyday expressions
- Progress tracking (local storage)
- Mobile-friendly responsive design
